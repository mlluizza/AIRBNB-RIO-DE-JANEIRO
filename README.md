# Projeto de Análise de Dados: Análise de Preços e Tendências do Mercado Imobiliário no Rio de Janeiro

### 1️⃣ Descrição do Projeto

Este projeto tem como objetivo a análise exploratória de dados (EDA) relacionados aos preços e características das listagens do Airbnb no Rio de Janeiro. A análise busca responder questões como:

- Quais fatores influenciam os preços dos imóveis?
- Como a sazonalidade afeta os preços ao longo do ano?
- Quais bairros têm as melhores e piores médias de preço?


### 2️⃣ Fontes de Dados Utilizadas
1. **Arquivo CSV - Dados do Airbnb**: 
    - **Descrição**: Contém informações sobre preços, localização, e outros atributos.
    - **Tipo de Dados**: Estruturados (dados tabulares)
    - **Método de Coleta**: O arquivo foi importado para o projeto diretamente via upload.
    - **Link do Kaggle**: [Kaggle Dataset]
      ([https://www.kaggle.com/datasets](https://www.kaggle.com/datasets/allanbruno/airbnb-rio-de-janeiro))

### 3️⃣ Análise de Dados
A análise de dados foi conduzida com as seguintes etapas:
 - Análise descritiva para explorar as principais distribuições e padrões.
 - Identificação de correlações entre variáveis como vendas, categoria de produto e região.

1. **Limpeza e Pré-processamento dos Dados**
 - Remoção de valores ausentes ou inconsistentes.
 - Conversão de formatos de datas e valores monetários para facilitar a análise.
   
2. **Análise Exploratória de Dados (EDA)**
 - Visualização de distribuições de preços por bairro e tipo de imóvel.
 - Identificação de correlações entre preços e características como número de quartos e localização.

3. **Identificação de Insights**
 - Padrões de preços ao longo dos meses do ano.
 - Comparação de preços médios por tipo de imóvel e bairro.

### 4️⃣ Ferramentas Utilizadas
- **Python** (pandas, matplotlib, numpy, matplotlib, seaborn)
- **Looker Studio** (para criação de dashboards e visualizações interativas)
