# Projeto de Análise de Dados: Análise de Preços e Tendências do Mercado Imobiliário no Rio de Janeiro

### 1️⃣ Descrição do Projeto

Este projeto tem como objetivo a análise exploratória de dados (EDA) relacionados aos preços e características das listagens do Airbnb no Rio de Janeiro. A análise busca responder questões como:

- Quais fatores influenciam os preços dos imóveis?
- Como a sazonalidade afeta os preços ao longo do ano?
- Quais bairros têm as melhores e piores médias de preço?


### 2️⃣ Fontes de Dados Utilizadas
link: https://www.kaggle.com/datasets/allanbruno/airbnb-rio-de-janeiro
